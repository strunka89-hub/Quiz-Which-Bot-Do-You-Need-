# Интеграция квиза в сообщество ВК

Квиз открывается по обычной ссылке в браузере (внутри ВК-приложения или
обычном мобильном браузере). Никаких VK Mini Apps, ничего устанавливать
не нужно — достаточно задеплоить проект и вставить ссылку в нужные места
сообщества.

Во всех примерах используется плейсхолдер `https://ВАШ-ДОМЕН.ru` — подставьте
реальный адрес, который вы получили после деплоя на Vercel/Netlify
(например, `https://quiz-ushakova.vercel.app`).

---

## 0. Ссылки с UTM-метками (рекомендуем использовать везде)

Чтобы в отчётах было видно, откуда пришёл лид, добавляйте UTM:

| Место размещения | Ссылка |
| --- | --- |
| Кнопка действия сообщества | `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=cta_button` |
| Меню сообщества | `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=menu` |
| Закреплённый пост | `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=pinned_post` |
| Автоответ в сообщениях | `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=autoreply` |

UTM-метки автоматически приедут вместе с лидом в поле `utm` в JSON.

Также удобно сократить ссылку через встроенный сервис ВК:
`https://vk.cc/` → вставьте туда полную ссылку с UTM, получите короткий
`vk.cc/XXXXXX`, который удобно использовать в постах и сторис.

## 1. Кнопка действия сообщества (главный вход)

Это самый видимый элемент на мобильной версии страницы ВК: большая синяя
кнопка под шапкой сообщества.

Путь:

1. Открыть `https://vk.com/ushakova_ai`.
2. Справа (или «…» на мобильном) → **Управление**.
3. Вкладка **Кнопка действия** (на мобильном: **Настройки → Кнопка действия**).
4. Включить опцию, в поле **Действие** выбрать **«Открыть сайт»**.
5. В поле **Текст кнопки** написать: `Пройти тест`.
6. В поле **Ссылка** вставить:

```
https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=cta_button
```

7. Сохранить.

Теперь каждый посетитель сообщества с мобильного видит большую кнопку
«Пройти тест» прямо под шапкой.

---

## 2. Меню сообщества

Закреплённые кнопки-ссылки под описанием группы.

Путь:

1. В сообществе → **Управление** → **Меню** → **Новая кнопка**.
2. **Тип**: внешняя ссылка.
3. **Адрес**: `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=menu`.
4. **Название кнопки** (макс. ~20 символов): `Пройти тест`.
5. **Обложка**: загрузить изображение 376×256 (можно использовать
   `public/og-cover.png` из проекта — он хорошо смотрится как превью).
6. Сохранить.

## 3. Закреплённый пост

Самый конверсионный формат для ВК — пост с крупным превью.

### Текст поста (можно копировать)

```
Тест для предпринимателей: где вы теряете клиентов?

5 коротких вопросов — 60 секунд — и вы получите персональный разбор:
где именно в вашем процессе теряются заявки и что можно автоматизировать.

Пройти тест → https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=pinned_post
```

### Как закрепить

1. Опубликовать пост от имени сообщества.
2. В верхнем правом углу поста → **«…» → Закрепить**.
3. Проверить, что рядом со ссылкой появляется **большое превью** с OG-картинкой.
   Если превью не подтянулось — через 5–10 минут обновите страницу и
   перезакрепите пост (ВК кэширует OG).

## 4. Автоответ в сообщениях сообщества

Чтобы каждый, кто пишет в группу, сразу получал ссылку на тест.

Путь:

1. **Управление** → **Сообщения** → **Настройки для бота**.
2. Включить **«Приветствие»** (первое сообщение от бота).
3. Текст приветствия:

```
Здравствуйте!
Вот тест, который покажет, где вы теряете клиентов 👇

https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=autoreply

5 вопросов · 60 секунд · сразу получите разбор.
```

4. Дополнительно можно добавить **клавиатуру бота** с кнопкой-ссылкой
   «Пройти тест» (там же, в настройках бота → «Клавиатура бота» → кнопка
   типа «Открыть ссылку»).

---

## 5. Сторис сообщества

1. В **ВК Админ** на мобильном → **Истории** → **Новая история**.
2. Добавить изображение (можно использовать `public/og-cover.png`).
3. Наложить стикер-ссылку → вставить
   `https://ВАШ-ДОМЕН.ru/?utm_source=vk&utm_medium=community&utm_campaign=story`.
4. Опубликовать.

---

## 6. Как проверить, что превью ссылки отображается

ВК подтягивает OG-теги из `index.html` и показывает большое превью
с изображением `og-cover.png`.

1. Откройте любой диалог ВК (можно с самим собой) → пришлите туда
   `https://ВАШ-ДОМЕН.ru/` и дождитесь, пока появится превью.
2. Если превью не подтянулось — подождите 1–2 минуты (ВК кэширует).
3. Альтернативно проверить OG-теги можно на сторонних сервисах:
   - `https://www.opengraph.xyz/` — вставьте URL и посмотрите, как будет
     выглядеть превью.

Если нужно принудительно обновить кэш ВК — добавьте к ссылке любой
параметр, например `?v=2`, и ВК перезапросит OG-теги.

## 8. Что делать, если лид не доехал

Даже если webhook временно упал, квиз сохраняет лид в `localStorage`
браузера пользователя (ключ `quiz-ushakova-ai:leads`) и показывает
экран «Спасибо». В консоли браузера видно `[quiz] VITE_WEBHOOK_URL не
задан, лид сохранён локально: {...}` — но у рядового клиента этот
fallback не сработает «в минус»: с точки зрения UX он получает ответ
мгновенно.

Рекомендация: следите за откликом webhook (n8n / Make умеют мониторить)
и дублируйте уведомление о новом лиде в Telegram или на e-mail.
